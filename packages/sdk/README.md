# @wafflefinance/sdk

Shared SDK for the WaffleFinance protocol. Provides TypeScript bindings and utilities for interacting with both Ethereum and Stellar (Soroban) HTLC contracts.

## Soroban Contract Schema

When interacting with or extending bindings for the Stellar Soroban `wafflefinance-htlc` contract, refer to the formal IDL and schema documentation:

[Soroban HTLC IDL Reference](../../soroban/docs/HTLC_IDL.md)

It contains essential information on account layouts, available data types (like `OrderStatus` and `Order`), and entrypoint parameters needed for SDK development.
